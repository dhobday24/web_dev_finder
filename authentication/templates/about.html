Welcome to Pitch

What is PITCH

PITCH is an online platform that helps musicians find gigs. A venue posts an event that require a musician and musicians can apply to play at that event

How to use Pitch effectively

    STEP 1: Update Profile

  • Once registered, Please go to update profile option on your homepage

    • STEP 2: Upload a profile picture

  • You must have a profile picture uploaded to your account

    • STEP3: Select User type

  • if you are a venue, please selece Venue from Type user drop down menu
  • if you are a musician, please selece Musician from Type user drop down menu
