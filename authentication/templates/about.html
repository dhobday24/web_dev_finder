PITCH is an online platform that helps musicians find gigs. A venue posts an event that require a musician and musicians can apply to play at that event.

How to use Pitch effectively

Once registered, Please go to your profile and click on update profile.

Select User type

If you are a Venue, please selece Venue from Type user down menu on the update profile page.

If you are a Musician, please selece Musician from Type user down menu on the update profile page.

Upload a Profile Picture

Let everyone see you.

Update your Location

Available events are based on your location
